How It Works

01

Upload CIF File

Provide a standard Crystallographic Information File (.cif) describing your material's atomic structure.

02

Structure Parsing

The backend uses pymatgen to parse the CIF, extracting lattice parameters, atomic sites, and space group symmetry.

03

T₂ Computation

A physics-based model computes the spin–spin coherence time T₂ based on the crystal structure and magnetic environment.

04

Results

Receive structured JSON output with formula, lattice constants, atom count, and computed T₂ value.
